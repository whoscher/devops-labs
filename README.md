# devops-labs
Система мониторинга сервисов (Bash + Docker + Cron) 
Практический проект по созданию легковесной системы мониторинга состояния процессов в Linux-средах.  

Возможности проекта: 
Универсальный мониторинг: Скрипт проверяет наличие любого процесса в системе (по умолчанию bash). 
Логирование: Автоматическая запись статусов (UP/DOWN) с временными метками для анализа стабильности. 
Контейнеризация: Полная упаковка в Docker на базе Ubuntu 25.10  
Автоматизация: Готовая конфигурация для планировщика Cron (запуск каждые 5 минут). 

Структура проекта 
monitor.sh - основной Bash-скрипт с логикой проверки через pgrep. 
Dockerfile - инструкция по сборке изолированного образа на базе Ubuntu. 
.gitignore - настроенная фильтрация для исключения логов и временных файлов из репозитория. 

Быстрый запуск (Docker) 
Сборка образа: docker build -t my-monitor . 
Запуск мониторинга процесса bash (по умолчанию): docker run --rm --name monitor-up my-monitor 
Запуск мониторинга конкретного сервиса (например, nginx): docker run --rm my-monitor /bin/bash /usr/local/bin/monitor.sh nginx 

Настройка автоматизации (Cron) 
Для запуска мониторинга на хост-машине добавьте задачу в crontab -e: */5 * * * * /полный/путь/к/monitor.sh > /dev/null
